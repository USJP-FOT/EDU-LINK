## Installation
### Prerequisites
Ensure you have the following installed:
- Node.js
- npm / yarn

### Steps
1. Clone the repository:
   ```sh
   git clone https://github.com/USJP-FOT/EDU-LINK.git
   cd EDU-LINK
   ```
2. Install frontend dependencies:
   ```sh
   cd frontend
   npm install
   ```
3. Start the frontend:
   ```sh
   npm start
   ```

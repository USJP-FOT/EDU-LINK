# EDU-LINK

## Overview
EDU-LINK is a mobile application that helps to manage classrooms effectively and efficiently.

## Features
- **Dashboard** ‚Äì check classroom status.
- **Student Registration** ‚Äì Add new students to the system.
- **Push-Notificaton** ‚Äì Send push notifications when the classroom is ready.
- **Send notices** ‚Äì Send emails to the students.

## Technologies Used
- **Frontend:** Expo (React Native)

## Installation
### Prerequisites
Ensure you have the following installed:
- Node.js
- npm / yarn

### Steps
1. Clone the repository:
   ```sh
   git clone https://github.com/USJP-FOT/EDU-LINK.git
   cd EDU-LINK
   ```
2. Install EDU-LINK dependencies:
   ```sh
   npm install
   ```
3. Start the EDU-LINK:
   ```sh
   npm start
   ```
